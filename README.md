# Python Data Cleaning Script

A Python script that cleans CSV data by removing empty rows and invalid entries.

## Features
- Reads data from a CSV file
- Removes empty rows
- Writes cleaned data to a new CSV file

## Technologies Used
- Python
- CSV module

## How to Run
1. Clone the repository
2. Ensure data.csv is in the same directory as clean_data.py
3. Run:
   ```bash
   python clean_data.py
